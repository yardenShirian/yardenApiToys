Toys API docomentation

Routes

All HTTP methods are supported. You can use http or https for your requests.

  • Get all toys/toys
  • Get all products by category:/toys/cat/:catname
  • Search products by name or info:/toys/search?s=(info or name)
  • Price range:/toys/price?min=number&max=number
